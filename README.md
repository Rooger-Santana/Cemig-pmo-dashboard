Este projeto foi desenvolvido como um estudo de caso para a gestão de ativos críticos no setor elétrico, focado em governança de comissionamento e indicadores de performance físico-financeira.



Problema Identificado

Projetos de engenharia de grande porte (como subestações da CEMIG) sofrem com a fragmentação de dados entre as equipes de campo e a diretoria, dificultando a visualização do Caminho Crítico e o impacto no DVA (Demonstração do Valor Adicionado).



Solução Proposta

Um dashboard interativo que consolida:

Gestão de Valor Agregado (EVM): Monitoramento de SPI (Prazo) e CPI (Custo).

Quadro Kanban Ágil: Controle visual das etapas de comissionamento.

Análise de Riscos: Identificação proativa de gargalos no fornecimento de equipamentos (ex: Transformadores).

Distribuição de Valor: Gráfico de DVA alinhado aos relatórios de transparência do setor elétrico.



Tecnologias Utilizadas

Frontend: HTML5, Tailwind CSS (Design Responsivo).

Dados: JavaScript & Chart.js (Visualização de dados em tempo real).

Metodologia: PMBOK 7ª Edição e Scrum.
